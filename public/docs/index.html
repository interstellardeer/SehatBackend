Sehat API Documentation

Clinics Management

GET /api/clinics

Get all clinics

POST /api/clinics

Create new clinic
Body: { Nama, Lokasi, Telepon, Jam_Operasional }

PUT /api/clinics/:id

Update clinic
Body: { Nama, Lokasi, Telepon, Jam_Operasional }

DELETE /api/clinics/:id

Delete clinic

Patients Management

GET /api/clinics/:klinik_id/patients

Get all patients for a specific clinic

GET /api/clinics/:klinik_id/patients/search?keyword=

Search patients in a clinic

Healthcare Workers (Nakes) Management

GET /api/clinics/:klinik_id/nakes

Get all healthcare workers for a clinic

POST /api/clinics/:klinik_id/nakes

Add healthcare worker to clinic
Body: { Nama, Spesialisasi, Nomor_STR, Jenis_Kelamin, Telepon, Email }

Visits Management

GET /api/clinics/:klinik_id/visits

Get all visits for a clinic

GET /api/clinics/:pasien_id/visits

Get all visits for a specific patient

POST /api/clinics/:klinik_id/visits

Add visit record
Body: { pasien_id, nakes_id, Tanggal_Kunjungan, Keluhan, Diagnosa }

Medical Records

GET /api/clinics/:klinik_id/medical-records

Get all medical records for a clinic

GET /api/clinics/:pasien_id/medical-records

Get all medical records for a specific patient

POST /api/medical-records

Create medical record
Body: { kunjungan_id, Detail_Rekam_Medis }

Payments

GET /api/clinics/:klinik_id/payments

Get all payments for a clinic

POST /api/payments

Create payment record
Body: { kunjungan_id, Total_Bayar, Metode_Bayar, Status_Pembayaran }

Authentication

POST /api/auth/login

User login
Body: { email, password }

POST /api/auth/register

User registration
Body: { Nama, Tanggal_Lahir, Jenis_Kelamin, Alamat, No_Telepon, Email, Password, klinik_id }

POST /api/auth/logout

User logout
